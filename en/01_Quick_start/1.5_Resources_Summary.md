## Software Development Resources

| Resource Type                    | Link | Description |
| -------------------------------- | ---- | ----------- |
| ⭐ AI Example Projects (Demo Zoo) | [Code Repository](https://gitee.com/bianbu/spacemit-demo.git) | Example projects developed by the Bianbu Robot team and optimized for SpacemiT chips, providing end-to-end inference demos.<br />- **CV branch**: Image classification, object detection, and face recognition in both C++ and Python.<br />- **NLP branch**: Multimodal voice interaction with ASR, LLM, and TTS modules, supporting voice-based Q&A. |
| Bianbu Development Guide         | [Guide Entry](https://www.spacemit.com/community/document/info?lang=en&nodepath=software/SDK/bianbu/root_overview.md) | Guidance on using the Bianbu SDK, developing Qt applications, building JavaScript frontends and Python backends, creating Docker images, and more. |
| Remote Connection Tools          | [MobaXterm Download](https://mobaxterm.mobatek.net/download.html) | An all-in-one terminal tool that supports SSH, SFTP, and serial access, making it useful for remote debugging of the Bianbu Robot SDK. |
