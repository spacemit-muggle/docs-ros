# Text-to-Speech (TTS)

## Function Introduction

TTS (text-to-speech) is a technology that automatically converts input text into audible speech. Through linguistic analysis, acoustic modeling, speech synthesis, and other steps, it generates fluent and natural human speech from natural language text. TTS technology is widely used in scenarios such as intelligent voice assistants, service robots, human-computer interaction, and accessible reading.

This example uses the SpacemiT AI computing core for TTS model inference and publishes the computation results via ROS 2 messages.

## Environment Setup

- It is recommended to use the ROS2_LXQT operating system.
- Make sure `source /opt/bros/humble/setup.bash` has been executed in all terminals by default.

### Hardware Connection

![](./images/tts1.png)

This example uses a `Wheeltec USB sound card + speaker` to verify the audio generated by TTS. Alternatively, you can use other USB speaker devices.

**Note:**

- The device should be plug-and-play under Linux ALSA/PipeWire without requiring additional drivers.
- The device should support at least two sampling rates, 44.1 kHz and 48 kHz, to ensure compatibility with common speech models and audio libraries such as PortAudio, PyAudio, and librosa.

### Call Service

**Terminal quick call:**

```
ros2 service call /tts_service jobot_interfaces/srv/TTSInfer "{text: '你好'}"
```

If the speaker outputs speech, the service is operating normally.

**Python code call**

```python
import rclpy
from rclpy.node import Node
from jobot_interfaces.srv import TTSInfer

class TTSClient(Node):
    def __init__(self):
        super().__init__('tts_client')
        self.cli = self.create_client(TTSInfer, 'tts_service')
        while not self.cli.wait_for_service(timeout_sec=1.0):
            self.get_logger().info('Service not available, waiting...')

    def send_request(self, text):
        req = TTSInfer.Request()
        req.text = text
        future = self.cli.call_async(req)
        rclpy.spin_until_future_complete(self, future)
        return future.result()

def main(args=None):
    rclpy.init(args=args)
    client = TTSClient()
    response = client.send_request('你好')
    print('Response:', response)  # According to TTSInfer definition, this may be audio path or byte data.
    client.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()

```
