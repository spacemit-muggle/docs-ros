# 上电开机

本章说明第一次拿到K3开发板后如何上电开机：完成供电、显示器接线、调试串口（可选），并在PC看到串口启动日志，确认板子与线缆工作正常。

## 1. 物品清单

| 类型 | 说明 |
| --- | --- |
| 开发板 | SpacemiT K3系列；这里以Com 260为例，见图。 |
| 供电 | DCIN 供电，19V/2.37A 或 12V/5A 电源适配器 |
| 显示器 | 带 DP、MIPI DSI 口 的显示器 |
| 键盘&鼠标 | 任意 USB-C 键盘/鼠标（鼠标支持蓝牙）
| 串口调试 | USB 转 TTL 刷机串口线（作为可选） |
| USB | Type-C |

串口线和显示器二选一，新手推荐使用显示器

![图1](assets/com260_apperance.jpg)


## 2. 接线顺序

新手如果没有串口调试，可以通过使用DP接屏幕的方式，以下主要演示使用显示的方式初始配置。如果没有显示器可以跳转按照第3章节接串口的方式。

![](assets/k3_connection.jpg)


1. 接入显示器 DP/MIPI，注意：**MIPI 显示器不支持热拔插，在上电前接入、下电后拔出**
2. 接入电源
3. 接入鼠标键盘
除了 MIPI 显示屏，以上步骤没有特定顺序

## 3. 串口连接

串口线接线图如下：

![按键](assets/com260_button.jpg)

务必对照 丝印 GND/RX/TX 与 USB-TTL 引脚接入

参考线序说明（仅当你的 USB-TTL 线与此一致时）：黑线接地、白线接板端 RX 丝印位、绿线接 TX 丝印位、红线多为电源 空置不接

串口接好后，上电就可以进入到终端。默认的用户名: bianbu或root，密码：bianbu


```bash
    nmcli device wifi connect "wifi_name" password "wifi_password"
```

## 4. 远程登录

设备连接上网络之后，就可以通过远程登录的方式接入。

### 4.1 ssh

- Windows 登录
    推荐使用 MobaXterm 工具，操作如下：

    1）打开 MobaXterm，点击 “Sessions” → “New Session”，选择 SSH。

    2）设置连接参数：

    Remote host：开发板 IP 地址（如 192.168.1.100）；
    Specify username：默认用户名为 bianbu；
    Port：保持默认 22。
    3）点击 OK 发起连接，并输入密码完成登录。


- Ubuntu 登录

    在终端中输入以下命令：
    ```
    ssh bianbu@<remote_ip>
    ```
    将 <remote_ip> 替换为开发板的实际 IP 地址。首次连接时会提示确认主机指纹，输入 yes 即可。

## 5. 常见问题

| 现象 | 建议排查 |
| --- | --- |
| 串口完全无输出 | 供电是否正常；RX/TX 是否交叉且 GND 已接；PC 是否选对 COM/设备节点；波特率是否为 115200。 |
| 乱码 | 波特率与板载 UART 约定不一致；尝试核对硬件手册中的调试口波特率；排除 USB 线松动或接触不良。 |
| 仅依赖线色接线 | **不可靠**；务必对照 **丝印 GND/RX/TX** 与 USB-TTL 模块引脚定义。 |
| 是否需要先烧镜像 | 出厂若已有固件，接线正确即可见日志；重新刷机或换镜像见 [2.2-镜像烧录](2.2-镜像烧录.md)。 |
