```

```

### 2.2 快速运行
下载模型资源
```
bash examples/yolov8/scripts/download_models.sh

bash scripts/download_assets.sh

```
运行
```
yolov8 examples/yolov8/config/yolov8.yaml
```

（可选）：使用摄像头实时检测
```
yolov8 examples/yolov8/config/yolov8.yaml --use-camera --camera-id 0  # 确认摄像头 id
```

## 4. Agent 智能控制

通过 Agent 框架实现自然语言控制机器人，Agent 会自动理解意图、调用工具、执行操作并返回结果。本节以 Hermes Agent 为例，演示如何配置 Agent 并通过 MLink 工具网关控制设备。

### 4.1 安装与配置 Agent

Agent 可选用 OpenClaw 或 Hermes Agent，推荐使用 Hermes Agent（体验更好）。

```bash
# 安装 Hermes Agent
sudo apt-get update && sudo apt-get install -y hermes-agent

# 配置模型（按向导配置密钥和模型）
hermes model

# 启动交互
hermes
```

### 4.2 配置 Robot Skills

Robot Skills 为 Agent 提供机器人领域的专业技能，安装后 Agent 可自动完成 SDK 获取、环境配置、编译、模型下载和性能测试等操作。

```bash
# 获取 Robot Skills
git clone https://github.com/spacemit-robotics/robot-skills.git

# 安装到 Hermes
mkdir -p ~/.hermes/skills/spacemit-robot
cp -rf robot-skills/skills/* ~/.hermes/skills/spacemit-robot/
```

验证安装：

```bash
hermes skills list
```

预期输出包含以下技能：

| 技能名称 | 技能组 | 来源 |
| :--- | :--- | :--- |
| spacemit-robot-llm | spacemit-robot | local |
| spacemit-robot-sdk-bootstrap | spacemit-robot | local |
| spacemit-robot-shared | spacemit-robot | local |

确认安装后，启动 Hermes 进行对话测试：

```bash
hermes
# 输入：请测试 spacemit llm 性能数据给我
```

Agent 会自动执行：获取 SDK 代码 → 配置环境 → 编译 → 下载模型 → 测试性能 → 返回结果。

更多的机器人技能，敬请持续关注。
